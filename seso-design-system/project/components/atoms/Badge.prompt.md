Small pill for type/status metadata — the green `H-2A` worker-type tag is the canonical use.

```jsx
<Badge tone="green">H-2A</Badge>
<Badge tone="neutral" dot>Draft</Badge>
<Badge tone="green" solid>Active</Badge>
```

Tones: `neutral` / `green` / `blue` / `amber` / `red`. `solid` for filled, `dot` for a leading status dot.
